## 💡 ABOUT THE PROJECT

This project serves as my digital business card and professional portfolio. It is designed with a mobile-first approach, ensuring a seamless user experience across all devices. The UI is inspired by modern vCard aesthetics, featuring a sticky sidebar for quick contact access and dynamic content tabs for smooth navigation.

## 🛠️ TECHNICAL ARSENAL

This portfolio was built from the ground up without heavy frameworks to ensure maximum performance and total control over the DOM.

*   **Markup & Styling:** Semantic `HTML5` and custom `CSS3` (utilizing CSS variables, flexbox, and grid for layout).
*   **Interactivity:** Vanilla `JavaScript` for modal toggling, smooth scrolling, and dynamic page routing.
*   **Typography & Iconography:** `Poppins` font family via Google Fonts and scalable vector icons via `Ionicons`[cite: 5].
